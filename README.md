# dynamic-tokenization-review

This repository accompanies our review paper on **dynamic tokenization methods in the Transformer era**. It contains the search protocol, methodological details, and review logs used to support transparency and reproducibility.

## Repository contents

- `PROTOCOL.md`: Search protocol and methodological details of the review.
- `SEARCHLOG.md`: Search log and documentation of the literature search process.
- `RAYYANLOG.md`: Screening log and Rayyan-related documentation.
- `search_log/`: Supporting files related to the search process.
- `rayyan_log/`: Supporting files related to the screening process.

## Representative dynamic tokenization implementations

To improve the practical usefulness of the review, this repository also provides a curated list of representative dynamic tokenization implementations and public code resources across modalities. These links are intended to help readers connect the taxonomy and conceptual discussion in the paper with concrete implementations.
